## ✨ Features

- ✅ Classic **Tic-Tac-Toe** gameplay
- ✅ Multiplayer with **Unity Netcode for GameObjects**
- ✅ Simple **lobby system** (Host / Client)
- ✅ **Data synchronization** between players
- ✅ Lightweight project – great for learning networking basics
